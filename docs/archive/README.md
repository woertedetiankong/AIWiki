# Archive

This directory preserves historical AIWiki design and implementation notes that
are no longer part of the root documentation set.

Root documentation is now intentionally small:

- `README.md`: user-facing setup, quick start, and command reference.
- `AGENTS.md`: engineering rules for AI coding agents.
- `SPEC.md`: current implementation contract.
- `SPEC-FUTURE.md`: optional backlog and future contracts.
- `prd.md`: concise product direction.
- `CHANGELOG.md`: milestone summary.

Archived files:

- `prd-full-2026-04-29.md`: full historical PRD before documentation compaction.
- `llm-wiki.md`: original background essay / product inspiration.
- `implementation-m*.md`: detailed milestone implementation records.

Treat archived files as historical context. If an archived note conflicts with
code, `README.md`, `SPEC.md`, or `prd.md`, prefer the current root documents and
the code.
